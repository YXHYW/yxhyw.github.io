如何阅读一本书

  • 第一篇：阅读的层次

阅读的活力与艺术​

太多的资讯就如同太少的资讯一样，都是一种对理解力的阻碍​

  1. 信息爆炸的年代，垃圾信息比有效信息更多，如何避免被垃圾信息充斥覆盖？
  2. 接收垃圾信息会浪费大量时间。

阅读是一件主动的事，阅读越主动，效果越好​

  1. 阅读时该如何主动？我的方式是做笔记、做书摘、思考。
    1. 阅读的目标：为获得资讯而读，以及为求得理解而读。
    2. 此处暂时设置阅读的三个层次目标：获得娱乐；获得资讯；获得理解。
  2. 不同的目标，决定何种阅读方式。
    1. 简单的阅读艺术：这是一个凭借着头脑运作，除了玩味读物中的一些字句之外，不假任何外助，以一己之力来提升自我的过程。
    2. 阅读还有其他目标，如娱乐。
    3. 要能在阅读中被作者启发，一要知其言，二要知其意，三要知其何出此言。
    4. 被启发的重点在于不要止于吸收资讯。
    5. 读得多不一定读得好，正如蒙田所说：“初学者的无知在于未学，而学者的无知在于学后。”总有些书呆子，读的太广，却读不通。
    6. 不同于以往的“多读书，读好书。”在这信息爆炸的年代，我们应该“读好书，多读好书，精读好书。”
    7. 其实，任何学习都不该没有活力，就像任何阅读都不该死气沉沉。
    8. 学习分两类：指导型学习和自我发现型学习，或者说，辅导型和非辅导型学习。
    9. 阅读时的思考，包括：敏锐的观察、灵敏可靠的记忆、想象的空间、训练有素的分析、省思能力。

阅读的层次​

一般来说，阅读的规则是：努力越多，效果越好。（越是细嚼慢咽，消化就越好。）

第一层次的阅读：基础阅读。这一层次的阅读关注“作者在说什么。”

第二层次的阅读：检视阅读。特点在强调时间。在有限的时间里，最完整的阅读。

第三层次的阅读：分析阅读。重点在追求理解。在无限的时间，最好最完整的阅读。

第四层次的阅读：主题阅读。

阅读的第一个层次：基础阅读​

学习阅读四个阶段：

  1. 阅读准备阶段，大致水平相当于幼儿园，要求视力良好，保持注意力等。
  2. 阶段二：认字，掌握最初级的读写能力。
  3. 阶段三：词汇增长，能轻易阅读简单文体。
  4. 阶段四：精进与增进前面所学技巧，能消化阅读经验。

阅读的第二个层次：检视阅读​

检视阅读两个阶段：

  1. 检视阅读一：有系统的略读或粗读*
    1. 先看书页名，然后如果有序就先看序；
    2. 研究目录页；
    3. 如果书中附有索引，也要检阅一下；
    4. 如果有出版者介绍，不妨读一下；
    5. 开始挑几个看起来跟主题息息相关的篇章来看；
    6. 最后一步，把书打开，东翻翻西翻翻，念个一两段，有时候连续读几页，但不要太多。不要忽略最后两三页。
    7. 目标：应了解这本书是否包含想要继续挖掘的内容，是否值得继续投下时间和精力，能被归为哪一个种类。***
  2. 检视阅读二：粗浅的阅读
    1. 头一次面对一本难读的书的时候，从头到尾先读完一遍，碰到不懂的地方不要停下来查询或思索。
  3. 阅读的速度
    1. 理想上来说，不只是要能读得快，还要能用不同的速度来阅读。
  4. 补充：
    1. 在第一次阅读一本难度的书时，不要企图了解每一个子句。

如何做一个自我要求的读者​

  1. 主动阅读时要提出问题，并尝试去回答
    1. 整体来说，这本书再谈些什么？
    2. 作者细部说了什么，怎么说的？
    3. 这本书说的有道理吗？是全部有道理，还是部分有道理？
    4. 这本书跟你有什么关系？
  2. 以记笔记的方式让一本书真正属于自己
    1. 在书上直接做笔记，不必心疼书；
    2. 检视阅读时做一下结构笔记；
    3. 分析阅读时做一下概念笔记。
  3. 养成阅读的良好习惯
    1. 由许多规则中养成一个习惯
